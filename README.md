# dy-player

## 安装依赖
```
npm install
```

### 开发环境
```
npm run serve
```

### 打包
```
npm run build
```

### 调试服务[网易云接口原仓库地址 仅供学习使用](https://binaryify.github.io/NeteaseCloudMusicApi/#/)
```
npm run proxy
```

### electron下调试
```
npm run start （注意此时不需要开启调试服务）
```
